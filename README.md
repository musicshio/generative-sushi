# これはもう寿司ってことになりませんかね？

https://generative-sushi.vercel.app/O

[Qiita クソアプリ Advent Calendar 2025](https://qiita.com/advent-calendar/2025/kuso-app) シリーズ2の21日目、「[掛布団を敷布団に乗せればそれはもう寿司ってことになりませんかね？？？](https://zenn.dev/music_shio/articles/fa69b9388fdf00)」のアプリです。

## Getting Started

```bash
bun install
bun dev
```

##  Tech Stacks

- Framework: Next.js (App Router) / TypeScript / Bun
- Style: DaisyUI
- Deployment: Vercel
- Storage: Vercel Brob / Vercel Postgres
- ORM: Prisma (Vercel Integration)
- AI: Vercel AI SDK / AI Gateway
  - Image Generation
  - Generative User Interfaces

## Deployment
`main` ブランチにmergeするとvercelが自動でデプロイします。
